# Git_相关知识总结

## 1.  “git pull”、 “git fetch” 和 “git clone”的区别

+ **git clone** : 

  + 自动将远程主机命名为origin，拉取它的所有数据（复制到本地）。

  + 创建一个指向它的master分支的指针，并且在本地将其命名为**origin/master**。

  + 创建一个与origin的master分支在指向同一个地方的本地master分支。
